Class SimpleLexer

Description
Description | Descendents | Vars (details) | Methods (details)

Accepts text and breaks it into tokens.

Some optimisation to make sure the content is only scanned by the PHP regex parser once. Lexer modes must not start with leading underscores.

Located in /php_parser.php (line 180)
